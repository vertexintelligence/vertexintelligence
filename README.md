## ⚡ GENX / RAINBOW Architecture

A two-layer sovereign AI platform:

| Layer | Purpose | Visibility |
|---|---|---|
| 🌈 **RAINBOW** | Local sovereign execution — memory, security, incident replay, performance tuning | Private-by-default |
| ⚡ **GENX** | Public intelligence interface — APIs, dashboards, docs, integrations | Public surface |

**Core Principle:** RAINBOW must remain sovereign. GENX can fail gracefully. All upgrades prove in RAINBOW before release into GENX.

## 🔐 Governance Architecture

Operating under **Two-Key Governance Law** — a dual-control merge protocol:

- No direct pushes to main — all changes require PR and approval
- Hard compile gates: make lint-all must pass before any push
- Self-approval is prohibited — engineering and review authority are separated
- Zero secrets in GENX — secrets are purged and never committed
